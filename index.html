About this app

Instructions for using the app:

  • 3-key interface:
    • Press J key to select previous element
    • Press K key to select next element
    • Press G key to click the element.
  • 5-key interface:
    • Press W key to select up
    • Press A key to select left
    • Press S key to select down
    • Press D key to select right
    • Press G key to click the element.

Test description: I compared the the 3-key and 5-key versions of the interface.

Example problems:

  • 93/12= Clear
  • 78*34= Clear
  • 0+5= Clear
  • 16-9= Clear
  • 72*58= Clear
